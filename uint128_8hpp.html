Detailed Description

Unsigned integer type with width of exactly 128 bits.

Author
Vladimir Shaleev
Copyright
MIT License

The library adds its own type uint128_t, because it is missing in the standard. The library does not use compiler extensions such as __int128 for greater portability.

Algorithms from the implementation of UInt128 for .NET are taken as a basis
